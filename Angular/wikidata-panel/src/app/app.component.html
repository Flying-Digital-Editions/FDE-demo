Flying Digital Editions

ATTENTION: This is just a demo of the official Flying Digital Editions project! Check out the main repository here.

Flying Digital Editions is a project that has the aim of creating digital editions in a dynamic and simple way, to give the user a new way of exploring text, linking entities automatically to a knowledge base and retrieving information. The idea is to give users the opportunity to upload any type of texts, which will be analysed for creating a digital edition on-the-fly. Automatically, the tool will extract the most important entities from the text and link them to the Wikidata knowledge base, assigning an ID. The user can check the assigned IDs to ensure that the entities are linked correctly and can decide to delete or add other entities. Once saved, an interface will present a summary table of entities by category on the left and text with the entities highlighted in the centre. By clicking on them, an information panel will appear on the right with all the basic information.

For more information, go to the demo repository.
